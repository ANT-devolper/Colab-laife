# Climate survey

> **Status:** 🚧 planned — not implemented yet.

## Purpose

Run organizational climate / engagement surveys and report on the results.

## Key concepts / entities

- **Survey** — a set of questions issued to employees over a period.
- **Response** — an employee's (often anonymous) answers.
- **Result / report** — aggregated outcomes per survey.

## Main flows

- Create and launch a survey to a target audience.
- Employees respond (anonymity rules defined at design time).
- Aggregate and view results; surface highlights on the dashboard.

## Permissions / roles

- Admins/HR create surveys and view aggregates; individual responses respect the survey's
  anonymity rules.
